# Vocabulário Inglês para Crianças

Uma ferramenta interativa para ensino de vocabulário inglês para crianças, com jogos de memória e correspondência de palavras.

## Características

- Interface amigável e colorida para crianças
- Jogos interativos: jogo da memória e jogo de correspondência de palavras
- Categorias de vocabulário:
  - Frutas
  - Cores
  - Números (0-10, 0-20, 0-100)
  - Animais da fazenda
  - Animais do zoológico
- Design responsivo para uso em diferentes dispositivos
- Funciona offline, sem necessidade de conexão com internet

## Como Usar

1. Abra o arquivo `index.html` em qualquer navegador moderno
2. Escolha uma categoria de vocabulário
3. Selecione um dos jogos disponíveis:
   - Jogo da Memória: encontre pares de palavras e imagens correspondentes
   - Jogo de Palavras: arraste palavras para suas imagens correspondentes

## Estrutura do Projeto

```
├── index.html           # Página principal
├── css/

├── js/

    └── zoo-animals/
```

## Requisitos Técnicos

- Navegador web moderno com suporte a JavaScript ES6
- Não requer instalação ou conexão com internet
- Funciona em dispositivos móveis e desktops

## Desenvolvimento

Este projeto foi desenvolvido usando HTML5, CSS3 e JavaScript puro, sem dependências externas, para garantir o funcionamento offline.
